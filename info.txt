Que el alumno practique el flujo completo de trabajo en GitHub partiendo desde cero: crear un repositorio nuevo, clonarlo, trabajar en una rama, realizar cambios y generar un Pull Request (PR).
¿Que es un Pull Request? Un pull request (PR) es una solicitud en plataformas como GitHub o GitLab para fusionar cambios de código de una rama a otra, generalmente de una rama de desarrollo a la principal, permitiendo que el equipo revise, discuta y apruebe las mejoras antes de integrarlas. Es fundamental para la colaboración, asegurando la calidad del código.
